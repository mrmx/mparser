.About MParser:
------------

MParser is a XML parser targetted to be lightweight and with constant/very low memory greedy, thus is focused
to be used within constrained environments such as J2ME/CLDC v1.0.


.Contents:
----------

Under docs/ folder you can find JavaDoc html documentation.
Under lib/ folder:

  MParser_SE: Compiled MParser full version with sax1 extension and tests. (Compiled without debugging info and optimized)
  MParser_ME: Compiled MParser with minimum classes (no sax1 extension neither test classes), suitable to J2ME - Preverified code -

Under src/ folder: Complete MParser source code :)

If you want to get the newest Version of MParser please visit http://www.mrmx.org/mparser
